Scuba Lizard

A 2D underwater adventure featuring a lizard protagonist. Created in 48 hours during the Global Game Jam 2025. I did not create the art for this project, I coded the game systems, and applied animations to premade sprite sheets. My biggest strenght in this project was being able to use chatgpt to work beyond my own personal skill for coding in C++. Additionally I acted as a project manager for this, and was able to effectively delegate tasks and work through challenges in that 48 hours.

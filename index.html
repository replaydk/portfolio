Featured Projects

The Jungle Demo

A low-poly first-person horror game set in the jungle. This was my first major project as a game designer, beginning with asset packs, level design, and an introduction to node-based coding. It also features exceptional audio design. Originally developed for my Game Engines class, I continued refining it throughout Game Studio. Future updates will introduce a tutorial system, an inventory system, revamped level design, and a deeper narrative with puzzles and interactive elements.

Scuba Lizard

A 2D underwater adventure featuring a lizard protagonist. Created in 48 hours during the Global Game Jam 2025. I did not create the art for this project, I coded the game systems , and applied animations to premade sprite sheets. My biggest strength in this project was being able to use ChatGPT to work beyond my own personal skill for coding in C++. Additionally, I acted as a project manager for this and was able to effectively delegate tasks and work through challenges in that 48 hours.
